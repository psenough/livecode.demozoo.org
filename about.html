About

The Demoscene

The demoscene is an international computer art subculture focused on producing demos: self-contained, sometimes extremely small computer programs that produce audiovisual presentations.

Some references to learn more about the demoscene:

  • Wikipedia article
  • Teach Yourself Demoscene in 14 Days

Live Coding

Live Coding is a contributive performance with one or multiple artists using code to generate real-time graphics and/or music.

Shader Showdown

Two performers, during 25 minutes and with the same starting screen, sit down and code from memory something that will make the audience vote for them.

Usually done using a tool called Bonzomatic.

Shader Jam

Multiple live coders just jamming some visuals for a music act/DJ set.

Shader Royale

Bigger number of live coders (not just two as in a Shader Showdown), starting at same time, getting eliminated by the audience votes in a Battle Royale format.

Byte Battle

Similar to Shader Showdown, but using the fantasy console TIC-80, and limited to 256 bytes is code length.
